# HazeBase Controllers & Accessories

Source: https://hazebase.com/en/controls/

---

## Commander — Cable Remote Control

- **Type:** Wired cable remote control
- **Connection:** 5-pin XLR connector
- **Compatible with:** All HazeBase models with 5-pin connector
- **Features:**
  - Latch function (sustained operation without holding button)
  - Ready display (visual status indicator)
- **Note:** Supplied without cable — cable sold separately
- **Manuals:** Available in DE and EN

---

## Communicator — Radio Remote Control

- **Type:** Wireless radio remote control
- **Connection:** 5-pin with Ready indicator
- **Compatible with:** All HazeBase Generation 2 machines
- **Features:**
  - Wireless operation
  - Ready status indicator
- **Made in Germany**

---

## XLR Remote — Cable Remote Control

- **Type:** Wired cable remote control
- **Connection:** 3-pin XLR socket
- **Compatible with:** HazeBase fog machines and hazers (check individual machine specs)
- **Note:** Supplied without cable
